# Keyboard Shortcuts

## Dashboard Viewer

| Shortcut | Action |
|----------|--------|
| **Alt + Right Arrow** | Next dashboard (alphabetical) |
| **Alt + Left Arrow** | Previous dashboard |
| **F11** | Toggle browser fullscreen |
| **Escape** | Exit fullscreen |

Dashboard switching is disabled while in edit mode.

## AI Builder

| Shortcut | Action |
|----------|--------|
| **Enter** | Send message |
| **Shift + Enter** | New line in message |

## Modals & Forms

| Shortcut | Action |
|----------|--------|
| **Escape** | Close modal (may prompt if unsaved changes) |
| **Enter** | Submit / confirm in most modals |
